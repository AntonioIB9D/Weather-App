# Weather App 🌦️

🛠️ Aplicación para consultar el clima 

Acciones disponibles en la aplicación:

1. Consultar el clima de una ciudad. ⛈️


### Notas adicionales 📗

##### Para utlizar la aplicación se deberan:

---

Instalar modulos de node ⬇️ :

```
npm install
```

Ejecutar la aplicación ⚙️ :

```
npm run dev
```
